# Business Problem

Fleet operators and business stakeholders need better visibility into trip demand, driver earnings, and operational performance to optimize business decisions.

---

# Project Overview

This interactive Power BI dashboard analyzes New York City For-Hire Vehicle (FHV) trip data and transforms raw operational data into business insights.

The dashboard enables users to:

- Monitor overall trip activity
- Analyze demand trends
- Evaluate driver earnings
- Compare operational performance
- Identify peak operating periods
